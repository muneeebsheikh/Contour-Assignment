# Contour-Assignment
React Redux Todo List

To start the app:

Download or clone the repo.

then follow these commands:

using npm
 - cd ./todo-app
 - npm install
 - npm install @mui/material @emotion/react @emotion/styled 
 - npm start
 
using yarn:
 - cd ./todo-app
 - yarn install
 - yarn add @mui/material @emotion/react @emotion/styled 
 - yarn start
